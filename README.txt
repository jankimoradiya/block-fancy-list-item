== Description ==

This Plugin is use for extend gutenberg core listing block.

So, It provide diffrent type of list style setting interface. You can change the different icon of list item as well change the text properties.


== Installation ==

 To install this plugin please follow these steps:

1. Upload `block-fancy-list-item` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Select List Block from block list.
